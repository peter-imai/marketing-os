# Marketing OS

**A starter operating system for your work, built on [Claude Code](https://claude.com/claude-code).**

Most people use AI like a chat: you ask, it answers, you copy the result somewhere else — and the next conversation starts from zero. Nothing accumulates.

This is the opposite. It's a system that works directly on your real files and gets sharper every time you use it, because the work *is* the context. You set up a workflow once, and it runs the way you would. Do that across the things you repeat, and one person starts producing like a team.

It grew out of thousands of hours running marketing for real companies, so the worked examples lean marketing — but the structure is general. Whatever you do that repeats, this is a place to start handing it to an agent that already knows your context.

## Getting started

You'll need [Claude Code](https://claude.com/claude-code) installed. Then:

1. **Open this folder in Claude Code.** This folder *is* your system — everything lives here.
2. **Run `/start`.** It sets up your output style, your profile, and your first workspace, explaining each step as it goes.

That's the whole setup. For the full walkthrough — what to do in your first session and the habits that make the system compound — see **[START-HERE.md](START-HERE.md)**.

## Make it yours

This is a base, not a finished product. Fork it, delete what you don't need, and build components one at a time as you work. It's meant to be adapted.
